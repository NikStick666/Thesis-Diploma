Our History

As part of the completion of the final project for the 121 “Software Engineering” program, the primary focus was on developing the software architecture and writing the code for the web application. To ensure a modern and user-friendly experience (UX), the graphical interface of Volvo's official website was chosen as a visual reference for the client-side. I independently implemented this interface from scratch (writing markup, styles, and scripts).
